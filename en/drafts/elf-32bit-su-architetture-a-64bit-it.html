Prima

gambuzzi@vq19:~/bin/polygen-1.0.6$ ./polygen
-bash: ./polygen: No such file or directory

I comandi da dare

sudo dpkg --add-architecture i386
sudo apt-get update
sudo apt-get install libc6:i386 zlib1g:i386
